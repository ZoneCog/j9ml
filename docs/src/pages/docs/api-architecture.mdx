## Data Flow Through APIs

Understanding how data flows through Jan's API layers is crucial for effective development and debugging.

```mermaid
sequenceDiagram
    participant Client
    participant API_Gateway
    participant Auth
    participant Controller
    participant Service
    participant Extension
    participant AI_Engine
    participant Storage

    Note over Client,Storage: API Request Flow

    Client->>API_Gateway: HTTP Request
    API_Gateway->>Auth: Validate Token
    Auth-->>API_Gateway: Token Valid
    API_Gateway->>Controller: Route Request
    Controller->>Service: Process Request
    Service->>Extension: Extension Hook
    Extension->>AI_Engine: Inference Request
    AI_Engine->>Storage: Load Model Data
    Storage-->>AI_Engine: Model Response
    AI_Engine-->>Extension: Inference Result
    Extension-->>Service: Processed Result
    Service-->>Controller: Service Response
    Controller-->>API_Gateway: HTTP Response
    API_Gateway-->>Client: JSON Response

    Note over Client,Storage: WebSocket Stream Flow

    Client->>API_Gateway: WebSocket Connect
    API_Gateway->>Controller: Stream Request
    Controller->>Service: Initialize Stream
    Service->>Extension: Start Streaming
    loop Inference Chunks
        Extension->>AI_Engine: Process Chunk
        AI_Engine-->>Extension: Chunk Result
        Extension-->>Service: Stream Chunk
        Service-->>Controller: Forward Chunk
        Controller-->>API_Gateway: WebSocket Message
        API_Gateway-->>Client: Stream Data
    end
    Extension-->>Service: Stream Complete
    Service-->>Controller: End Stream
    Controller-->>API_Gateway: Close Stream
    API_Gateway-->>Client: WebSocket Close
```

## API Integration Patterns

Common patterns for integrating with Jan's API architecture.

### 1. **Direct REST Integration**
```typescript
// REST API client example
const janClient = new JanApiClient('http://localhost:1337');

// Load a model
await janClient.models.load('llama-3-8b');

// Send chat completion
const response = await janClient.chat.completions.create({
  messages: [{ role: 'user', content: 'Hello!' }],
  model: 'llama-3-8b'
});
```

### 2. **WebSocket Streaming**
```typescript
// WebSocket streaming example
const ws = new WebSocket('ws://localhost:1337/chat/stream');

ws.onmessage = (event) => {
  const chunk = JSON.parse(event.data);
  // Handle streaming response chunks
};
```

### 3. **Extension Development**
```typescript
// Extension API example
export class CustomExtension extends BaseExtension {
  async onLoad() {
    // Register API endpoints
    this.registerCommand('custom_operation', this.handleCustomOperation);
    
    // Listen to system events
    this.events.on('model_loaded', this.onModelLoaded);
  }
  
  private async handleCustomOperation(params: any) {
    // Custom business logic
    return { success: true, data: params };
  }
}
```

This comprehensive API architecture enables flexible integration while maintaining security, performance, and maintainability across all system components.
